# CampusFace — Reconnaissance faciale
from django.db import models
import face_recognition

class Student(models.Model):
  name = models.CharField(max_length=200)
  face_encoding = models.BinaryField()
  attendance = models.ManyToManyField(
    'Attendance', blank=True)

  def verify_presence(self, frame):
    matches = face_recognition.compare_faces(
      [self.face_encoding], frame)
    return any(matches) # ✓

À propos

Qui suis-je ?

Je suis Kambale Mukovi Volonté VD, développeur passionné originaire de la République Démocratique du Congo 🇨🇩, actuellement étudiant à l'UNILUK (Université Libre des Pays des Grands Lacs, Nord-Kivu).

Ma passion pour le code est née d'un désir de résoudre des problèmes réels dans mon environnement. Que ce soit une plateforme de gestion scolaire ou un système de reconnaissance faciale, je construis des solutions qui ont un impact concret.

"Le code est mon langage, l'innovation mon moteur. Chaque ligne que j'écris rapproche ma communauté d'un monde meilleur."

En dehors du développement, je m'intéresse à l'électronique embarquée avec Arduino, à l'IA appliquée et à l'open source. Je crois fermement que la technologie doit être un levier d'émancipation pour l'Afrique.
